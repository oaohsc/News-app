1. **Open the `.env` file** in the root directory (`c:\News app\.env`)

2. **Replace the placeholder values** with your actual API keys:

```env
VITE_NEWS_API_KEY=your api keys
VITE_OPENAI_API_KEY=your api keys
```

3. **Save the file**

4. **Restart the dev server**:
   - Stop the current server (Ctrl+C in the terminal)
   - Run `npm run dev` again

## Getting API Keys

- **News API**: https://newsapi.org/ (free tier available)
- **OpenAI API**: https://platform.openai.com/api-keys (requires account and credits)
